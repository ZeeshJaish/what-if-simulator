# What If – Behavioral Pattern & Outcome Simulator

## Overview
What If is a web-based application that helps users understand how their daily habits influence long-term outcomes. It allows users to input routines such as study time, sleep, and savings, and simulates possible future results based on those patterns.

## Problem Statement
People often follow daily habits without clearly understanding their long-term effects. There is a lack of simple tools that connect present behavior with future outcomes in an interactive way.

## Objective
To build an application that analyzes user habits, identifies patterns, and simulates future results using visual representations.

## Features
- Input daily habits such as study hours, sleep, and savings  
- Basic pattern analysis (consistency and discipline)  
- Simulation of long-term outcomes  
- Scenario comparison (current vs improved habits)  
- Graph-based visualization  
- Real-time updates using React  

## Tech Stack
- HTML  
- CSS  
- JavaScript  
- React.js  
- Chart.js / Recharts  

## How It Works
1. User enters habit-related inputs  
2. System applies rule-based logic  
3. Patterns are analyzed  
4. Future projections are generated  
5. Results are displayed using charts  

## Use Cases
- Estimating savings growth  
- Understanding impact of study habits  
- Comparing lifestyle improvements  

## Future Scope
- User authentication  
- Advanced prediction models  
- More customizable simulations
